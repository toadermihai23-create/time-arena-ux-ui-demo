# time-arena-ux-ui

UI playground minimal pentru TimeArena (Nikita, 11 ani).
Scop: testăm rapid UI/UX + structura jocului, cu logică minimă.

## Features (v1)
- Mesaj zilnic + status bar (Max min, min câștigate, XP, Level, Streak)
- Navigație: Panou, Misiuni, Magazin, Penalități, Reguli, Istoric
- Misiuni pe categorii (5–7/categorie) + penalty corespondent
- Penalități pe niveluri (Lv 0–6) + countdown pentru banurile reale (Lv 3+)
- Magazin cu ticket-uri (cheltuie minutele câștigate)
- Istoric acțiuni (localStorage)

## Instalare local
Deschide `index.html` cu Live Server (recomandat) sau direct în browser.

## GitHub Pages
Settings → Pages → Deploy from branch → `main / root`
